
# Pesquisa no Mercado de Capitais
## Sobre o Projeto 

Plataforma web voltada à organização e consulta de pesquisas em Mercado de Capitais. O projeto auxilia pesquisadores iniciantes na identificação do estado da arte da literatura, reunindo estudos recentes, oportunidades de pesquisa e informações obtidas por meio de Revisões Sistemáticas de Literatura (RSL)

## Acessar Sistema Online

Acesse a aplicação:

🔗 [https://williamdovale.github.io/pesquisa-mercado-capitais/
](https://william-vale.github.io/Trabalho_T-picos_Mercado_Financeiro/)

Não é necessário instalar nenhum programa.
Basta abrir o link acima em qualquer navegador.

- HTML5
- CSS3
- JavaScript
- Node.js
- OpenAlex API
