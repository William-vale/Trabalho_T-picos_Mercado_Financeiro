# Pesquisa no Mercado de Capitais

## Sobre o Projeto

A plataforma "Pesquisa no Mercado de Capitais" foi desenvolvida para auxiliar pesquisadores, estudantes e demais interessados na identificação do estado da arte da literatura científica em Mercado de Capitais.

O sistema realiza buscas em bases acadêmicas e apresenta artigos científicos recentes, organizados por data de publicação, facilitando a descoberta de tendências e oportunidades de pesquisa.

---

## Como Executar o Sistema

### Requisitos

- Windows
- Node.js instalado

Download:

https://nodejs.org

---

## Instalação

### 1. Baixe o Projeto

Clique em:

Code → Download ZIP

Extraia os arquivos em uma pasta do seu computador.

Exemplo:

C:\PesquisaMercadoCapitais

---

### 2. Executar Automaticamente

Abra a pasta do projeto.

Dê dois cliques no arquivo:

abrir_sistema.bat

O Windows poderá solicitar permissão de execução.

Ao permitir:

✅ O servidor local será iniciado.

✅ O navegador será aberto automaticamente.

✅ O sistema estará disponível em:

http://localhost:3000

---

## Arquivo abrir_sistema.bat

```bat
@echo off

cd /d "%~dp0"

start http://localhost:3000

node server.js

pause
```
