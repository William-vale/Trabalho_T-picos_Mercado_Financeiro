# Trabalho_T-picos_Mercado_Financeiro
Plataforma web voltada à organização e consulta de pesquisas em Mercado de Capitais. O projeto auxilia pesquisadores iniciantes na identificação do estado da arte da literatura, reunindo estudos recentes, oportunidades de pesquisa e informações obtidas por meio de Revisões Sistemáticas de Literatura (RSL)
